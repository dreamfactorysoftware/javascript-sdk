getTables

List the names of the available tables in this storage. Use 'include_properties' to include any properties of the tables.

URL

https://yourdspname.cloud.dreamfactory.com/rest/mongohq

HTTP Method

GET

Response Type

Tables

Parameters

    include_properties : boolean (query)
    optional

    Return all properties of the tables, if any.

names : string (query)
optional

Comma-delimited list of the table names to retrieve.

body :  
Tables (body)
optional

Array of tables to retrieve.

getRecords

Use the 'ids' or 'filter' parameter to limit resources that are returned. Use the 'fields' parameter to limit properties returned for each resource. By default, all fields are returned for all resources. To send the 'ids' or 'filter' as posted data, use the POST command and set 'method' parameter to 'GET'.

HTTP Method

Response Type

    filter : string (query)
    optional

    SQL-like filter to limit the resources to retrieve.

    limit : int (query)
    optional

    Set to limit the filter results.

    offset : int (query)
    optional
